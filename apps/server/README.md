## Description

두디스 프로젝트 API 서버입니다.

## Installation

```bash
//프로젝트 루트에서
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm dev:back-end

# watch mode
$ pnpm dev:back-end

# production mode
$ npm run start:prod
```
